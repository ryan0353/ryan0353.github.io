HTML tags I've learned.

<h1> - <h6>
Heading tags used to define titles and subtitles. h1 is the biggest heading, with it decreasing with each interval.
<p>
Defines a paragraph of text.
<a>
Creates hyperlinks that connect pages or external websites.
<img>
Displays an image on the webpage using the src(source)attribute.
<abbr>
Defines an abbreviation and shows the full meaning when hovered over.
